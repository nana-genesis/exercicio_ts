# Exercício TypeScript - Funções de Multiplicação e Saudação
##  Descrição

Implementação de duas funções em TypeScript seguindo boas práticas de desenvolvimento:

1. **`multiplicacao(a: number, b: number): number`**  
   Retorna o produto de dois números com validações de segurança.

2. **`saudacao(nome: string): string`**  
   Retorna a string `"Olá [nome]"` com sanitização contra XSS.
